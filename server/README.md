# React-native-server

> 

## Getting Started

1. [NodeJS](https://nodejs.org/) and [npm](https://www.npmjs.com/).
2. Install dependencies

    ```
    npm install
    ```

3. Start app

    ```
    npm start
    ```

## Testing

Run `npm test` and all your tests in the `test/` directory will be run.
